SmartGarden

Este trabalho foi desenvolvido em parceria entre uma instituição educacional de ensino básico e a universidade virtual do estado de São Paulo (Univesp) que visa a realização de app de gestão hídrica em irrigação. Trazendo para a sala de aula uma solução inteligente de gestão hídrica de uma horta, este trabalho irá utilizar a inserção tecnológica como uma metodologia ativa de ensino.

O objetivo geral deste trabalho é realizar uma aplicação em IoT, que seja utilizada por uma comunidade escolar. Sendo este projeto em IoT integrado a um aplicativo que seja acessível aos discente da instituição. Ele se guia pelo objetivo exigido pela instituição para o Projeto Integrador III, sendo ele: “Desenvolver um software um aplicativo que utilize banco de dados, inclua script web (Java script), nuvem, acessibilidade, controle de versão, integração contínua e testes. Incluir um dos seguintes itens: uso e fornecimento de API, análises de dados e IoT.”

É interessante salientar que a utilização de tecnologias como metodologia educacional ativa é uma grande aliada à docência e a gestão hídrica (visando assim os conceitos de sustentabilidade) é um tema de extrema importância para o ensino, assim este trabalho irá unir estes dois fatores de maneira acessível. O aplicativo irá então realizar de maneira inteligente a quantidade de água de irrigação de plantas tomando como referência o nível de umidade da terra.
